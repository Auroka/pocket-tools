# Pocket Tools 设计系统

## 设计关键词
- 简单
- 简洁
- 安静
- 高效
- 轻盈

## 用户场景
- 在电脑端快速打开一个工具并立刻开始使用。
- 在移动端需要时也能直接访问，不依赖复杂交互或深层配置。
- 在计时、看时间、记录提醒这类高频短时任务中获得低干扰体验。

## 颜色系统
- `Background / #F6F7FB`：页面主背景，偏冷白，保证长时间观看舒适。
- `Surface / #FFFFFF`：主要卡片背景。
- `Surface Muted / #EEF2F6`：弱层级背景、输入框、分段控件底板。
- `Text / #121826`：主文本，强调清晰可读。
- `Text Muted / #667085`：辅助信息、说明文本。
- `Primary / #4F6BFF`：主操作色。
- `Primary Soft / #E8EDFF`：主色弱化背景。
- `Success / #22C55E`：完成态、正向反馈。
- `Warning / #F59E0B`：提醒与轻警告。
- `Danger / #FF5A5F`：风险操作与错误反馈。

## 字体系统
- 字体栈：`Inter, PingFang SC, Segoe UI, sans-serif`
- 页面标题：`28/34`，中高字重。
- 模块标题：`18/24`，中等字重。
- 正文：`16/24`
- 辅助说明：`14/20`
- 时间数字：`56-72`，启用 `tabular-nums`

## 间距系统
- 基于 4pt 栅格。
- 页面边距：`16-20`
- 卡片内边距：`16-20`
- 模块间距：`12-16`
- 大区块间距：`24-32`

## 圆角与阴影
- 小圆角：`14`
- 常规圆角：`20`
- 大圆角：`28`
- 阴影使用轻量柔和投影，避免厚重层叠。

## 按钮
- 主按钮：高 `52`，大圆角，主色填充，用于唯一核心行为。
- 次按钮：高 `52`，浅底或描边，用于重置、返回、辅助操作。
- 图标按钮：尺寸 `44`，用于次级快捷动作。
- 交互动效：`180ms`，只做轻微位移和阴影变化。

## 输入框
- 高 `52`
- 浅底 + 细描边
- 聚焦时显示单层清晰 `focus ring`
- 文本左对齐，数字输入启用更强可读性

## 卡片
- 采用浅色卡片承载信息。
- 结构优先为：标题 → 状态/说明 → 操作。
- 卡片间靠间距分层，不依赖粗边框。

## 导航模式
- 首页承担唯一分发入口，使用卡片或列表进入各个工具。
- 每个工具页面保持独立，不做工具间底部切换导航。
- 在桌面端优先保证信息密度和可读性，在移动端保证可点击和可滚动。

## 图标
- 使用 `24px` 线性圆角图标。
- 激活态可使用浅色胶囊背景承托。
- 图标语义明确，避免抽象装饰性图形。

## 入门风格
- 首次进入不做重 onboarding。
- 首页直接展示可用工具与状态，首屏即可点击进入。
- 通知权限只在需要提醒的场景按需请求。

## 高级视觉方向
- 以冷白背景为基底，局部叠加极轻渐变光晕。
- 使用少量玻璃感和柔雾高光提升精致度。
- 保持内容对齐和留白秩序，让视觉高级感来自排版与比例。
