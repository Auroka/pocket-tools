# 待办记录

## 工具说明
待办记录用于快速新增和管理临时事项，适合在电脑前处理工作任务时做轻量提醒和状态跟踪。

## 当前能力
- 新增待办事项
- 删除待办事项
- 标记待办完成或恢复为未完成
- 为待办设置固定时间提醒
- 为待办设置按分钟间隔重复提醒
- 在浏览器支持时发送系统通知，并提供页面内弹层提醒

## 提醒规则
- `固定时间提醒`：在指定时间触发一次提醒。
- `间隔提醒`：从创建待办开始计算，每隔指定分钟数重复提醒，每次到点都会发送新的浏览器通知和页面内提醒，直到待办完成或被删除。
- 已完成待办不会继续提醒。

## 适用场景
- 电脑端快速记下临时待办
- 需要每隔一段时间提醒自己跟进某件事
- 需要在固定时间收到一次提醒，但不想打开复杂任务管理工具

## 注意事项
- 待办数据保存在当前浏览器 `localStorage` 中，第二天重新打开同一浏览器、同一路径仍然可以看到之前的待办。
- 如果清理浏览器缓存、更换浏览器、更换访问域名或使用隐私模式，历史待办可能无法保留。
- 浏览器通知依赖权限和系统支持；若通知不可用，页面仍会弹出站内提醒。
- 提醒检查依赖页面处于打开状态。
