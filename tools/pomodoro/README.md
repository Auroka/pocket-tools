# 番茄时钟

## 工具说明
番茄时钟用于帮助用户在电脑前进行专注工作与休息切换，适合需要控制工作节奏、减少分心的场景。

## 当前能力
- 提供 `专注 / 短休息 / 长休息` 三种模式
- 支持开始、暂停、继续、重置计时
- 支持自定义各模式时长和长休息间隔
- 记录当天完成的番茄数量
- 在浏览器支持时发送系统通知，并提供声音提醒

## 适用场景
- 电脑端专注工作
- 学习、写作、编码等需要节奏管理的任务
- 临时需要一个本地即可使用的轻量计时工具

## 注意事项
- 通知能力依赖浏览器权限和系统支持。
- 当天完成数量保存在浏览器本地存储中，切换浏览器或清除缓存后会丢失。
- 当前页面保持独立使用，不与其他工具做页面内切换。
