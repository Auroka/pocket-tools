# pocket-tools

一个以电脑使用场景为主、同时兼容移动端的静态小工具集合。

## 快速打开
- 首页：[Pocket Tools](https://auroka.github.io/pocket-tools/)
- 番茄时钟：[打开工具](https://auroka.github.io/pocket-tools/tools/pomodoro/index.html)
- 时间显示：[打开工具](https://auroka.github.io/pocket-tools/tools/clock/index.html)
- 待办记录：[打开工具](https://auroka.github.io/pocket-tools/tools/todos/index.html)

## 当前工具
- `番茄时钟`
- `时间显示`
- `待办记录`

## 目录
- `index.html`：工具首页
- `tools/`：工具页面
- `src/styles/`：共享设计系统
- `src/data/`：工具配置
- `docs/`：架构与设计文档

## 约束
- 统一使用原生 `HTML/CSS/JS`
- 统一两个空格缩进
- 统一使用 `UTF-8` 编码
- 工具以独立页面形式组织
